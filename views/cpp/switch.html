Switch Case Statement.

The switch statement helps in testing the equality of a variable against a set of values.
Each value under comparison is known as a case.

Syntax

switch (variable)
{

case 1:
break;
case 2:
break;
default:

}

Example

#include
using namespace std;


int main()
{
int x = 20;
switch (x)
{

case 10:
cout<<"X is 10"; break;

case 20:
cout << "X is 20"; break;

case 30:
cout << "X is 30"; break;

default:
cout<<"X is not 10, 20 or 30"; break;

}

return 0;
}
