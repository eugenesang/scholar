C++ Strings: strcpy(), strcat(), strlen(), strcmp().

What is a String?

A string is a sequence of characters. A C++ string is an object of the std::string class.
The characters are stored sequences of bytes with access to a single character byte allowed.
C++ strings allocate memory dynamically.
More memory can be allocated to the string during run time if needed.

String Functions:

strcpy()

This is the string copy function. It copies one string into another string.
Syntax:
strcpy(string1, string2);
The two parameters to the function, string1 and string2, are strings. The function will copy the string string1 into the string 1.

strcat()

This is the string concatenate function. It concatenates strings.
Syntax:
strcat(string1, string2);
The two parameters to the function, string1 and string2 are the strings to be concatenated.
The above function will concatenate the string string2 to the end of the string string1.

strlen()

This is the string length function. It returns the length of the string passed to it as the argument.

Syntax:
strnlen(string1)
The parameter string1 is the name of the string whose length is to be determined.
The above function will return the length of the string string1.

strcmp()

This is the string compare function. It is used for string comparison.
Syntax:
strcmp(string1, string2);
The above function will return 0 if strings string1 and string2 are similar,
less than 0 if string1string2.
