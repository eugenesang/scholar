What are Strings?

A string in literal terms is a series of characters. eg.
ROSE.

syntax
//String is an array of characters
char[] arrSample = {'R', 'O', 'S', 'E'};
String strSample_1 = new String (arrSample);

Now we always cannot write our strings as arrays; hence we can define the String in Java as follows:

//Representation of String
String strSample_2 = "ROSE";

If a number is quoted in “ ” then it becomes a string, not a number anymore. That means
String S1 ="The number is: "+ "123"+"456";
System.out.println(S1);
then it will print: The number is: 123456
