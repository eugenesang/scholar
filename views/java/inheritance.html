Inheritance in Java OOPs

What is Inheritance?

Inheritance is a mechanism in which one class acquires the property of another class.

Types of Inheritance.

  1. Single Inheritance

    2. In Single Inheritance one class extends another class (one class only).

  2. Multiple Inheritance

    3. In Multiple Inheritance, one class extending more than one class. Java does not support multiple inheritance.

  3. Multilevel Inheritance:

    4. In Multilevel Inheritance, one class can inherit from a derived class. Hence, the derived class becomes the base class for the new class.

  4. Hierarchical Inheritance:

    5. In Hierarchical Inheritance, one class is inherited by many sub classes.

Syntax

class subClass extends superClass
{
//methods and fields
}
